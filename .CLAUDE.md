# Reglas OBLIGATORIAS
Para CUALQUIER interacción relacionada, ya sea responder una pregunta del usuario, explicar un concepto, analizar código existente, generar código nuevo o modificar archivos, consultar SIEMPRE la documentacion de Claude Code que esta en https://docs.anthropic.com/llms.txt ANTES de responder o tocar el código. Esto aplica a todo el proceso: antes de contestarle al usuario, antes de analizar código y antes de escribir o modificar cualquier archivo.

**Razón:** Claude Code tiene breaking changes frecuentes entre versiones, por lo que mi conocimiento interno puede estar desactualizado respecto a la ultima versión. La documentación consulta las las mejores prácticas vigentes, así que prevalece sobre cualquier conocimiento previo que yo tenga.
